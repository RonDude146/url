Project Overview

URL Safety Checker is a web application designed to verify the safety of URLs before users visit potentially harmful websites.
Problem: Malicious URLs can lead to phishing attacks, malware infections, and data theft, posing significant security risks to internet users.
Solution: Integrates multiple security APIs (Google Safe Browsing and VirusTotal) to provide comprehensive threat detection with a user-friendly interface.
Target Users: General internet users, security-conscious individuals, educational institutions, and small businesses seeking to enhance their web browsing safety.
